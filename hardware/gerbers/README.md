# Gerbers

Gerber and drill files generated from the PCB layout for manufacturer submission.

## Files

| Extension | Layer |
|-----------|-------|
| `.gtl` | Top copper |
| `.gbl` | Bottom copper |
| `.gts` | Top solder mask |
| `.gbs` | Bottom solder mask |
| `.gto` | Top silkscreen |
| `.gbo` | Bottom silkscreen |
| `.drl` / `.exc` | Excellon drill file |
| `.gko` | Board outline |
